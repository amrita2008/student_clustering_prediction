Student Clustering Project

Cluster students using KMeans based on engagement, participation, and performance for personalized learning insights.

Dataset: xAPI-Edu-Data

Features: raised hands, resources visited, announcements, discussion, parent survey, absenteeism

Includes: data cleaning, scaling, clustering, PCA & silhouette visualizations, cluster profiling, and predicting new student clusters.

How to run: place dataset in the notebook folder, install dependencies via requirements.txt, and run student_clustering.ipynb.
